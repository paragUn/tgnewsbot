## Deployment
Deployment process as easy as possible:
Required software:

#### -  terminal for running bash scripts
#### -  docker
#### -  docker-compose

to deploy application, switch to needed branch and run bash script:
#### $bash start.sh 'your_bot_name' 'your_bot_token' 
#### (without quotes)
That's all.
